# Project Manifest: Hybrid System UI Library

An avant-garde component library and framework running natively on top of `shadcn/ui` and Tailwind CSS. It merges the logical order of Bauhaus and Minimalism with the structural gravity of Neo-Brutalism, the tangible micro-depth of Skeuomorphism, and the futuristic telemetry of Glassmorphic Cyberpunk.

## System Tech Stack
* **Framework:** Next.js (App Router / React)
* **Styling Platform:** Tailwind CSS v4+
* **Core Base:** `shadcn/ui` / Radix Primitives
* **Motion Framework:** `motion.dev`
